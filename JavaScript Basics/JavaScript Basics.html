JavaScript Basics

JavaScript is a cross-platform, object-oriented scripting language. JavaScript is extremely popular for a variety of reasons. It is a small and lightweight languagee allowing maximum flexibility for developers to take it in a bunch of different directions. JavaScript lives inside a host environemt (a web browser or Node server), it can be connected to the objects of these environments to provide programmatic control over them.

  • Variable Declaration JavaScript variables are containers for storing Data Value- Imagine a cup you fill with coffee, the cup holds the coffee, a variable holds a value. All JavaScript variables must be identified with unique names. These unique names are called identifiers.

  • Variable Assignment Assignment operators assign values to JavaScript- our cup can now have coffee poured in it, giving our variable a vlue to hold. The = assignment operator assigns a clue to a variable.

  • Data types are an important concept; to be able to operate variables you need to know the data type. There are six data types that are JavaScript Primitive: Boolean- true or false; null-null aka nothing; Number-42 or 3.1459; String- "Coding Dojo Rocks!"; Array- [1,'Coding,',2,'Dojo']; and Object- {first_name:'Jane',last_name:'doe'}
